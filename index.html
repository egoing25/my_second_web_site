이곳은 자바 스크립트를 하기전과 하면서 배운것들을 담으려는 공간이다.

목차

  • html에서 배운것
  • css에서 배운것
  • java script에서 배운것


컴퓨터 단축키

window키 + E :파일 탐색기 열기
window + alt + D:날짜와 시간을 표시/숨김
window+,:잠시 바탕화면 살피기.
window+Ctrl+F:장치검색
window+Ctrl+Q:빠른 지원 열기{화면공유, 원격제어등 가능}
window+Ctrl+space bar:이전에 선택한 입력 옵션으로 변경하기
window+Ctrl+V:빠른 설정의 사운드 출력 페이지를 염. {출력장치, 공간 사운드 및 볼륨 믹서에 대한 설정 포함}
window+D:바탕 화면 표시/숨기기

window+↓:활성 창 최소화하기
window+A:간이 설정{네트워크, 음량조절등}
window+I:설정 열기
window+L:컴퓨터 잠그기
window+M:모든 창 최소화하기
window+←:창을 화면 왼쪽에 맞추기
window+.또는 window+;:이모티콘 패널 열기
window,+:돗보기로 확대
window+prt sc:전체 화면 스크린샷 캡처
window+R:실행 대화 상자열기
window+shift+←:활성 창을 왼쪽 모니터로 이동합니다.
window+shift+M:최소화된 창 복원
window+shift+R:비디오를 녹화할 화면영역을 선택합니다.
